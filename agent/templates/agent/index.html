API Endpoints

Method Endpoint Description Auth
POST /api/token/ Login — get access & refresh tokens No
POST /api/token/refresh/ Refresh expired access token No
POST /api/research/ Ask a research question — runs agent JWT
GET /api/research/history/ Get all past queries and answers JWT
DELETE /api/research/delete/ Clear all research history JWT
GET /api/docs/ Interactive Swagger API documentation No

Agent Tools

🔍 web_search

Searches the web via Tavily API for current information and news.

🌐 url_reader

Fetches and extracts clean text content from any URL.

📖 wikipedia_tool

Gets Wikipedia summaries for factual and background topics.

✂️ summarizer_tool

Condenses long text into structured bullet points using LLM.
